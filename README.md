# Staff Foundry — Website

A responsive, single-page website for Staff Foundry — an operational execution and systems consultancy.

## Overview

This is a fully self-contained static website built with pure HTML, CSS, and JavaScript. No build tools or dependencies are required.

## Features

- Fixed navbar with scroll effect
- Animated hero section
- Interactive insights accordion
- Animated services grid
- Flagship case study with metrics
- Success stories carousel with modals
- "Why Us" image cards
- Final CTA section
- Fully responsive (mobile-friendly)

## How to Run Locally

Simply open `index.html` in any modern browser — no server required.

```bash
open index.html
```

Or serve it with any static file server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

## How to Deploy

### GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Click **Save** — your site will be live at `https://<your-username>.github.io/<repo-name>/`

### Netlify / Vercel

Drag and drop the project folder on [netlify.com/drop](https://app.netlify.com/drop) or connect your GitHub repo on [vercel.com](https://vercel.com).
